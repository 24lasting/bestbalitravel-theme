Professional tour booking WordPress theme for travel agencies and tour operators.

== Description ==

Best Bali Travel is a premium WordPress theme designed specifically for tour operators, travel agencies, and tourism businesses. Built with modern standards and optimized for conversions.

**Key Features:**

* Custom Tour Post Type with booking system
* Multi-language ready with full translation support
* WooCommerce compatible
* Elementor page builder integration
* Mobile-first responsive design
* Custom checkout system (no WooCommerce required)
* WhatsApp booking integration
* Demo content one-click import
* SEO optimized
* Fast and lightweight

**Perfect For:**

* Tour operators
* Travel agencies
* Hotel and villa rentals
* Activity providers
* Destination websites

== Installation ==

1. Upload the theme folder to `/wp-content/themes/` directory
2. Activate the theme through 'Appearance > Themes' menu
3. Navigate to 'Best Bali Travel > Dashboard' for theme setup
4. Import demo content via 'Best Bali Travel > Demo Import'
5. Customize via 'Appearance > Customize'

== Frequently Asked Questions ==

= Does this theme require any plugins? =

The theme works standalone. Optional recommended plugins:
- Elementor (for page building)
- WooCommerce (for payment processing)
- Contact Form 7 (for contact forms)

= Is this theme translation ready? =

Yes! The theme is fully translation-ready using WordPress i18n functions. POT file included.

= How does the booking system work? =

The theme includes a custom booking system that redirects to WhatsApp. For online payments, WooCommerce integration is available.

= Can I customize the colors? =

Yes, use the WordPress Customizer (Appearance > Customize) to change colors, fonts, and layout options.
